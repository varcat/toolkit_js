# 快速开始

## 安装

npm

```bash
npm i wsp-toolkit

```

unpkg

```html
<script src="https://unpkg.com/browse/wsp-toolkit/dist/wsp-toolkit.global.js"></script>
```

## 使用

```ts
// esmodule
import { isExist } from "wsp-toolkit";

// commonjs
const { typeOf } = require("wsp-toolkit");

// unpkg
window.toolkit.isExist("");
```
