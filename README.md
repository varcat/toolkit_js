# toolkit

Commonly used functions.

## Usage

```npm
npm i wsp-toolkit
```

```javascript
// browser
import { isExist } from "wsp-toolkit";

// node
const { typeOf } = require("wsp-toolkit");
```

## API

- addition
- currying
- identity
- isEmpty
- isExist
- isNumber
- isSafeNumber
- toArray
- toNumber
- typeOf
- unique
- LinkedList
